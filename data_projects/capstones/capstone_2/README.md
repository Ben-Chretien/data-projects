# Airline Delay Prediction

## Overview
This project predicts arrival delays using airline delay data and identifies key factors contributing to delays.

## Approach
- Data cleaning and preprocessing
- Feature encoding and scaling
- Model training (Linear, Ridge, Random Forest)

## Results
| Model | R² | RMSE |
|------|----|------|
| Linear Regression | 0.827 | 5192 |
| Ridge Regression | 0.827 | 5192 |
| Random Forest | 0.888 | 4179 |

## Conclusion
Random Forest performed best, indicating that non-linear relationships play a key role in predicting delays.
